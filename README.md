# 📅 스마트 스케줄러

구글 캘린더 수준의 고급 기능을 제공하는 웹 기반 스케줄 관리 애플리케이션입니다.

## ✨ 주요 기능

### 🎯 기본 기능
- **할일 관리**: 제목, 설명, 시작/종료 시간, 우선순위, 카테고리 설정
- **다양한 뷰**: 타임라인, 리스트, 캘린더, 주간, 월간 뷰
- **필터링**: 전체, 긴급, 중요, 오늘, 지연된 할일 필터
- **통계**: 완료율, 평균 지연시간, 생산성 인사이트

### 🚀 구글 캘린더 수준의 고급 기능

#### 1. 드래그 앤 드롭
- **시간대 이동**: 타임라인 뷰에서 할일을 다른 시간대로 드래그하여 이동
- **날짜 이동**: 캘린더 뷰에서 할일을 다른 날짜로 드래그하여 이동
- **시각적 피드백**: 드래그 중인 요소의 투명도 및 회전 효과

#### 2. 일정 크기 조절 (Resize)
- **시간 조절**: 할일의 상단/하단 핸들을 드래그하여 시작/종료 시간 조절
- **실시간 업데이트**: 조절 중 실시간으로 시간 변경 반영

#### 3. 빠른 추가
- **더블클릭 추가**: 타임라인이나 주간 뷰에서 더블클릭으로 새 할일 추가
- **스마트 위치 감지**: 클릭한 위치에 따라 자동으로 시간 설정
- **즉시 편집**: 추가 후 바로 편집 모달 열기

#### 4. 일정 충돌 감지
- **자동 감지**: 새로운 할일 추가 시 기존 일정과의 충돌 자동 감지
- **경고 메시지**: 충돌 시 사용자에게 알림 및 확인 요청

#### 5. 복사/붙여넣기
- **Ctrl+C**: 현재 편집 중인 할일 복사
- **Ctrl+V**: 복사된 할일을 현재 시간에 붙여넣기
- **자동 ID 생성**: 붙여넣기 시 새로운 ID 자동 생성

#### 6. 키보드 단축키
- **Ctrl+N**: 새 할일 추가
- **Ctrl+S**: 할일 저장
- **Ctrl+C**: 할일 복사
- **Ctrl+V**: 할일 붙여넣기
- **Delete**: 할일 삭제
- **Escape**: 모달 닫기

#### 7. 색상별 카테고리
- **업무**: 파란색 (#667eea)
- **개인**: 초록색 (#28a745)
- **건강**: 주황색 (#fd7e14)
- **학습**: 보라색 (#6f42c1)
- **사회활동**: 분홍색 (#e83e8c)
- **기타**: 회색 (#6c757d)

#### 8. 반복 일정
- **매일**: 일일 반복
- **매주**: 주간 반복
- **매월**: 월간 반복
- **매년**: 연간 반복

#### 9. 알림 설정
- **5분 전**: 5분 전 알림
- **15분 전**: 15분 전 알림
- **30분 전**: 30분 전 알림
- **1시간 전**: 1시간 전 알림
- **1일 전**: 1일 전 알림

## 🎨 뷰별 기능

### 타임라인 뷰
- 24시간 시간대별 할일 표시
- 드래그 앤 드롭으로 시간대 이동
- 더블클릭으로 빠른 추가
- 리사이즈로 시간 조절

### 리스트 뷰
- 할일 목록 형태로 표시
- 체크박스로 완료 상태 토글
- 편집/삭제 버튼
- 우선순위별 색상 구분

### 캘린더 뷰
- 월간 캘린더 형태
- 드래그 앤 드롭으로 날짜 이동
- 할일 개수 표시
- 오늘 날짜 하이라이트

### 주간 뷰
- 7일 x 24시간 그리드
- 시간대별 할일 배치
- 더블클릭으로 빠른 추가
- 드래그 앤 드롭 지원

### 월간 뷰
- 월간 개요
- 날짜별 할일 미리보기
- 클릭으로 상세 보기

## 📊 생산성 인사이트

- 지난 주 대비 변화율

- 생산성 향상 팁

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript (ES6+)**: 클래스 기반 구조
- **LocalStorage**: 데이터 영구 저장
- **Chart.js**: 통계 차트
- **Font Awesome**: 아이콘

## 🚀 시작하기

1. **파일 다운로드**: 모든 파일을 동일한 폴더에 저장
2. **브라우저 열기**: `schedule.html` 파일을 웹 브라우저에서 열기
3. **즉시 사용**: 초기 샘플 데이터가 포함되어 있어 바로 사용 가능

## 📱 반응형 디자인

- **태블릿**: 터치 친화적 인터페이스
- **모바일**: 최적화된 모바일 레이아웃

- **데이터 백업**: 브라우저 설정에서 데이터 백업 가능

모듈화된 클래스 구조로 새로운 기능 추가 용이
