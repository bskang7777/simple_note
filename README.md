# 📅 스마트 스케줄러

중요도 기반 인사이트가 있는 할일 스케줄 예약 시스템입니다.

## ✨ 주요 기능

### 📋 할일 관리
- **스마트 할일 추가**: 제목, 설명, 시작/종료 시간, 카테고리, 태그 설정
- **우선순위 시스템**: 긴급, 높음, 보통, 낮음 4단계 우선순위
- **반복 할일**: 매일, 매주, 매월, 매년 반복 설정
- **알림 기능**: 5분~1일 전 알림 설정

### 🎯 다중 뷰 지원
- **타임라인 뷰**: 24시간 시간대별 할일 배치
- **리스트 뷰**: 체크박스로 완료 관리
- **캘린더 뷰**: 월간 할일 일정 확인

### 📊 생산성 인사이트
- **실시간 통계**: 긴급/중요/완료 할일 수, 완료율
- **시간대별 분석**: 가장 생산적인 시간대 추천
- **우선순위별 성과**: 중요도별 완료율 분석
- **스마트 추천**: 개인화된 생산성 향상 팁

### 🔍 필터링 & 검색
- **상태별 필터**: 전체, 긴급, 중요, 오늘, 지연
- **날짜 네비게이션**: 이전/다음 날 이동, 오늘로 돌아가기
- **실시간 검색**: 제목, 설명, 태그로 검색

## 🚀 사용법

### 기본 조작
- **새 할일 추가**: `Ctrl+N` 또는 "새 할일" 버튼
- **할일 저장**: `Ctrl+S` (편집 모드에서)
- **모달 닫기**: `Esc` 키
- **검색**: `Ctrl+F`

### 할일 관리
1. **할일 추가**: "새 할일" 버튼 클릭
2. **정보 입력**: 제목, 설명, 시간, 카테고리, 우선순위 설정
3. **태그 추가**: 쉼표로 구분하여 태그 입력
4. **저장**: "저장" 버튼 또는 `Ctrl+S`

### 뷰 전환
- **타임라인**: 시간대별 할일 배치 확인
- **리스트**: 체크박스로 완료 상태 관리
- **캘린더**: 월간 일정 전체 보기

### 인사이트 확인
- **인사이트 버튼**: 우측 상단 "인사이트" 버튼 클릭
- **완료율 차트**: 주간 완료율 트렌드 확인
- **우선순위별 성과**: 중요도별 완료율 분석
- **스마트 추천**: 개인화된 생산성 향상 팁

## 🔧 설치 및 실행

1. **저장소 클론**
   ```bash
   git clone https://github.com/yourusername/smart-scheduler.git
   cd smart-scheduler
   ```

2. **브라우저에서 실행**
   ```bash
   # 로컬 서버 실행 (선택사항)
   python -m http.server 8000
   # 또는
   npx serve .
   ```

3. **브라우저에서 접속**
   ```
   http://localhost:8000/schedule.html
   ```

## 🌟 주요 기능 상세

### 스마트 할일 관리
- **직관적 입력**: 모달 기반 할일 추가/편집
- **시간 설정**: 시작/종료 시간 정확한 설정
- **카테고리 분류**: 업무, 개인, 건강, 학습, 사회활동, 기타
- **태그 시스템**: 쉼표로 구분된 태그 입력

### 우선순위 시스템
- **긴급**: 즉시 처리 필요한 할일 (빨간색)
- **높음**: 중요한 할일 (주황색)
- **보통**: 일반적인 할일 (파란색)
- **낮음**: 여유가 있는 할일 (초록색)

### 반복 할일
- **매일**: 동일한 시간에 매일 반복
- **매주**: 같은 요일, 같은 시간에 반복
- **매월**: 같은 날짜에 반복
- **매년**: 같은 날짜에 반복

### 알림 설정
- **5분 전**: 즉시 시작할 할일
- **15분 전**: 준비 시간이 필요한 할일
- **30분 전**: 준비가 필요한 할일
- **1시간 전**: 장거리 이동이 필요한 할일
- **1일 전**: 중요한 할일 사전 알림
